# Gold Stage Title Pro

브라우저에서 실행되는 고화질 황금 스파클 타이틀 애니메이션입니다.

## 실행 방법

`gold_stage_title_pro.html` 파일을 Chrome, Edge, Safari 등으로 열면 됩니다.
별도 설치가 필요 없습니다.

## 기능

- 한글/영문/여러 줄 문구 입력
- 황금 금속 그라데이션 텍스트
- 다중 블룸과 글로우
- 텍스트 내부 반짝이 질감
- 별 모양 스파클과 주변 입자
- 빛 스윕, 충격파, 렌즈 글로우
- 반짝임 강도 및 등장 속도 조절
- 전체 화면
- PNG 저장
- 고해상도 디스플레이 대응

## 단축키

- Enter 또는 Space: 다시 재생
- F: 전체 화면
- H: UI 숨기기
